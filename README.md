# 📘 Pokédex React Native

Uma Pokédex desenvolvida com React Native e Expo consumindo dados da PokeAPI. O aplicativo permite pesquisar Pokémon em tempo real por nome, exibindo automaticamente os resultados conforme o usuário digita.

## ✨ Funcionalidades

- 🔎 Busca dinâmica por nome
- ⚡ Atualização automática enquanto o usuário digita
- 🖼️ Exibição de imagem e nome do Pokémon
- 🚫 Bloqueio de pesquisa por ID
- 🎨 Interface moderna nas cores azul e verde
- 📱 Layout responsivo

## 🛠️ Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- PokeAPI

## 📦 Instalação

Clone o projeto:

```bash
git clone https://github.com/seu-usuario/nome-do-repositorio.git
```

Entre na pasta:

```bash
cd nome-do-repositorio
```

Instale as dependências:

```bash
npm install
```

ou

```bash
yarn
```

Inicie o projeto:

```bash
npx expo start
```
